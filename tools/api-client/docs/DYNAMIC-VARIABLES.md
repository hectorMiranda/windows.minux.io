# Dynamic variables reference

| Token | Resolves to |
|-------|-------------|
| `{{$timestamp}}` | Unix seconds |
| `{{$isoTimestamp}}` | ISO-8601 string |
| `{{$randomInt}}` | 0–999 |
| `{{$uuid}}` / `{{$guid}}` | UUID v4 |
| `{{$randomColor}}` | hex color |
| `{{$counter}}` | incrementing counter |
